## Context

Checkout abandonment was high on mobile. Analytics pointed to confusion around shipping options and sudden error states that forced users to start over. Stakeholders wanted a clearer step-by-step path without adding visual noise.

## Approach

I mapped the full journey on real devices, noted every validation and timeout, and rewrote copy so each screen had one primary action. Error messages stated what went wrong and how to fix it, inline with the relevant field. I worked with engineering to preserve form state when possible and to avoid blank screens during slow network responses.

## Outcome

The revised flow held steady completion rates in A/B tests while support tickets related to checkout errors fell. The UI stayed visually quiet—strong typography and spacing did most of the work—so the brand did not need a full redesign to see benefit.
